Bu klasör basit bir işletme web sitesi için hazırlanmıştır.
Kullanım:
1. Dosyaları GitHub veya Netlify gibi platformlara yükleyin.
2. index.html anasayfa olarak çalışır.
3. style.css tasarımı sağlar.
4. images klasörüne logonuzu ve görsellerinizi ekleyin.
5. One.com DNS ayarları üzerinden domain yönlendirmesini yapabilirsiniz.
